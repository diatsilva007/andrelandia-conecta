# Plataforma de Gestão e Visibilidade para o Comércio Local

Este projeto visa conectar e dar visibilidade aos comércios de Andrelândia/MG e região, facilitando a gestão, divulgação e interação entre comerciantes e clientes locais.

## Estrutura do Projeto

- `frontend/`: Aplicação React (Vite) para interface do usuário
- `backend/`: API Node.js (Express) com Prisma e PostgreSQL

## Como iniciar

1. Instale as dependências em cada pasta (`frontend` e `backend`)
2. Inicie o backend e o frontend separadamente

## Objetivo

Promover o comércio local, facilitar a gestão dos estabelecimentos e aproximar comerciantes e consumidores da região.
